## 待定 Agent 任务：GitHub 地址转学习文档

### 背景
用户在聊天中发送一个 GitHub 仓库地址，希望自动转换为可下载的学习文档。

### 目标交互（先记录，后续再细化）
1. 必须通过弹窗进行多轮交互（不是只在聊天气泡里问答）。
2. 先让用户在弹窗中选择输出文件格式（例如：Markdown / Doc）。
3. 再由模型先分析 GitHub 项目结构与核心内容。
4. 将分析得到的 2-3 个核心内容点回传给用户，由用户选择：
   - 只要某一项；
   - 或者全部都要。
5. 用户确认后生成对应格式结果文件。
6. 最终提供下载入口（用户点击即可下载文件）。
